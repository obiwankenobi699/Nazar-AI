# Nazar AI

## Purpose

AI-powered CCTV analytics and video search platform.

## Stack

Frontend:
- React
- TypeScript

Backend:
- Express
- MongoDB

AI:
- Python
- OpenCV
- YOLO
- CLIP

## Services

frontend/
- Dashboard
- Search UI
- Alerts

backend/
- APIs
- Authentication
- Event storage

ai-services/
- Detection
- Tracking
- Search
- Anomaly detection

## Rules

- Use TypeScript in frontend/backend.
- Use Python in AI services.
- Reuse existing code before creating new modules.
- Keep services independent.
- Avoid unnecessary dependencies.
